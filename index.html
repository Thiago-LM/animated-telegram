📋 Introdução ao Git

Bem-vindo à nossa aula de exemplo! Abaixo estão alguns dos comandos mais importantes do Git.

1. Iniciando um repositório

Para transformar a sua pasta atual em um repositório Git, utilize:

git init

2. Verificando o status

Para ver quais arquivos foram criados ou modificados:

git status

3. Preparando os arquivos (Staging)

Para preparar os arquivos para serem salvos (adicionar todos):

git add .

4. Salvando as alterações (Commit)

Para registrar o snapshot no histórico do projeto:

git commit -m "Sua mensagem explicativa do commit"
